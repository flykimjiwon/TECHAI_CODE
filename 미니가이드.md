# 택가이코드 미니가이드

> 단일 바이너리 AI 코딩 어시스턴트. 설치 → 실행 → 질문. 끝.

---

## 1분 시작

```bash
# macOS
sudo cp techai-darwin-arm64 /usr/local/bin/techai && sudo chmod +x /usr/local/bin/techai

# Windows: C:\techai\techai.exe 복사 후 PATH 등록

# 실행
techai
```

첫 실행 시 API 키 입력 → 바로 사용 가능.

---

## 동작 방식

```
사용자 입력
    ↓
[Auto-Prefetch] 프롬프트에서 파일 경로 감지 → 자동 읽기
    ↓
[Knowledge] 질문 키워드 → 내장 74개 문서 중 매칭 → 컨텍스트 주입
    ↓
[LLM API] 시스템 프롬프트 + 도구 정의 + 히스토리 → 모델에 전송
    ↓
[Tool Calls] 모델이 도구 호출 (file_read, grep, file_write 등)
    ↓
[결과] 파일 수정 완료 → 사용자에게 응답
```

## 3개 모드 (Tab 전환)

| 모드 | 역할 | 특징 |
|------|------|------|
| **Super** | 만능 | 질문, 코드 수정, 분석 — 기본 모드 |
| **Deep Agent** | 자율 코딩 | 최대 100회 도구 반복, 질문 없이 알아서 |
| **Plan** | 계획 수립 | 읽기 전용, 실행 전 계획만 |

## 핵심 슬래시 명령어

### 프로젝트

| 명령어 | 설명 |
|--------|------|
| `/init` | 프로젝트 스캔 + .techai.md 생성 + Knowledge Pack 자동 선택 |
| `/compact` | 대화 히스토리 압축 (토큰 절약) |
| `/clear` | 대화 초기화 |
| `/undo` | 마지막 파일 수정 되돌리기 |
| `/diff` | 현재 세션에서 변경한 파일 diff 보기 |

### 세션

| 명령어 | 설명 |
|--------|------|
| `/sessions` | 저장된 세션 목록 |
| `/save` | 현재 세션 저장 |
| `/export` | 대화를 마크다운으로 내보내기 |

### 메모리

| 명령어 | 설명 |
|--------|------|
| `/remember 내용` | 프로젝트별 메모리 저장 (AI가 매번 참고) |
| `/forget` | 메모리 삭제 |

### 유틸리티

| 명령어 | 설명 |
|--------|------|
| `/copy` | 마지막 AI 응답 클립보드 복사 |
| `/commands` | 전체 명령어 목록 |
| `/help` | 도움말 |

## Knowledge Packs (지식 문서)

```bash
techai → /init
# → .techai.md에 knowledge_packs: react, database, terminal 자동 생성
```

수동 설정: `.techai.md` 편집

```markdown
knowledge_packs: react, database, java    # 필요한 것만 ON
knowledge_packs: none                     # 전부 OFF
# 줄 삭제 → 74개 전부 로드
```

내 문서 추가: `.tgc/knowledge/`에 `.md` 파일 넣기

## 구조도

```
techai (~25MB 단일 바이너리)
├── internal/
│   ├── app/          ← TUI + 메인 루프
│   ├── llm/          ← API 통신 (streaming + reasoning_content)
│   ├── tools/        ← 15개 도구 (grep, file_write, apply_patch 등)
│   ├── knowledge/    ← 지식 문서 인덱싱 + 검색
│   └── ui/           ← 채팅 렌더링 + 색상 구분
├── knowledge/docs/   ← 74개 내장 문서 (go:embed)
└── Makefile          ← make build / make build-release
```
