# 택가이코드 v0.6.0 — 주요 기능

AI 코딩 에이전트. 단일 바이너리, 설치 불필요.

## 핵심
- **3모드**: 슈퍼(만능) / 개발(코딩특화) / 플랜(분석전용) — Tab 전환
- **13개 도구**: 파일 읽기/쓰기/편집, 셸 실행, 검색, Git, 진단
- **SSE 스트리밍**: 실시간 응답, 토큰 속도 표시

## v0.6.0 신규
- **세션 저장**: SQLite 기반, /sessions로 이전 대화 복원
- **Git 통합**: 상태바 브랜치 표시, git_status/diff/log 도구
- **컨텍스트 압축**: 3단계 자동 압축 (긴 대화도 끊김 없음)
- **자율 모드**: /auto — AI가 독립적으로 작업 수행 (최대 20회)
- **커맨드 팔레트**: Ctrl+K 퍼지 검색
- **메뉴**: Esc 키 빠른 접근
- **해시 편집**: stale-edit 방지 안전 편집
- **코드 진단**: Go/TS/JS/Python 자동 감지 린터
