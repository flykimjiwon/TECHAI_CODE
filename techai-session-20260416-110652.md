### 제가 할 수 있는 일
| 분야 | 주요 기능 |
|------|-----------|
| **프로젝트 분석** | 현재 디렉터리 구조를 살피고, 언어·프레임워크·핵심 파일을 파악해 요약합니다. |
| **코드 작성·수정** | 파일을 읽고(`file_read`), 새 파일을 만들고(`file_write`), 기존 파일을 안전하게(`file_edit`, `hashline_edit`) 수정합니다. |
| **디버깅·테스트** | `shell_exec` 로 빌드·테스트 명령을 실행하고, `diagnostics` 로 린트·정적 검사 결과를 제공합니다. |
| **Git 작업** | `git_status`, `git_diff`, `git_log` 로 현재 상태를 확인하고, 커밋·푸시 등도 실행할 수 있습니다. |
| **검색·검색** | `grep_search`, `glob_search` 로 코드·리소스를 찾아보고, `knowledge_search` 로 사내·내장 지식(예: **BXM 프레임워크**, **Spring**, **React/Next.js**, **SQL**, **Shell** 등) 을 빠르게 찾아드립니다. |
| **멀티‑에이전트** | 코드 생성 → 검토 → 합의를 자동으로 진행해 품질을 높일 수 있어요. |
| **대화·설명** | 한국어로 친절히 설명하고, 필요하면 단계별 진행 계획을 알려드립니다. |
